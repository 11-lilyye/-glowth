# 成长可视化

一个本地网页，用于**课程学习进度**（含作业/视频/练习与 GitHub 上传）、**攀岩与训练记录**（含视频链接或本地上传）、**安排计划**，以及**数据表格导出**。

## 功能

### 课程进度（如 CS50x、CS50p）
- **录入课程**：课程名称、课程安排说明（每周计划、截止日期等）。
- **自我查看课程安排**：每门课下方展示安排说明。
- **作业 / 视频 / 练习**：为每门课添加多条项目，类型选「作业」「视频」「练习」，填写名称。
- **完成情况**：每条可勾选「已完成」。
- **是否上传 GitHub**：每条可勾选「已上传 GitHub」。
- 进度由「已完成条数 / 总条数」自动计算，柱状图 + 环形图展示。

### 攀岩与训练
- **记录类型**：攀岩（难度、结果）或 训练（训练内容、时长）。
- **视频**：可填视频链接（B 站、YouTube 等），或点击「选择本地视频」上传，视频保存在本机浏览器（IndexedDB），建议 &lt; 50MB。
- **攀岩/训练安排**：单独列表，可添加计划日期、类型、内容，方便安排下次攀岩或训练。

### 数据表格
- 切换「课程进度表」「攀岩/训练记录表」「攀岩/训练安排表」。
- 一键导出 CSV，可用 Excel 打开。

数据保存在浏览器本地（localStorage + IndexedDB），无需服务器。

## 使用

用浏览器直接打开 `index.html` 即可；或本地起一个静态服务：

```bash
# 若已安装 Python 3
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

```bash
# 若已安装 Node.js 和 npx
npx serve .
```

在浏览器中打开对应地址即可使用。
